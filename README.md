# StockFlow WMS

Warehouse inventory management system — Myanmar grocery products.

## Features
- Dashboard with stock analytics and charts
- Stock in / stock out with barcode scanning
- Smart barcode matching (ignores last 4 digits)
- Fuzzy product name search
- Category & brand management
- WhatsApp order request alerts to owner
- Role-based access (Admin / Operator / Owner)

## Deploy
Connected to Vercel — pushes to `main` branch auto-deploy.

## Local
Open `index.html` directly in any browser — no server needed.
